# How to Add Events to Database

## Run the seed script:

```bash
cd backend
node seedEvents.js
```

This will add 16 professional college events including:
- 3 AI & Machine Learning events
- 3 Hackathon events  
- 6 Technical workshops
- 4 Non-Technical events

All events have:
✅ Professional descriptions
✅ High-quality images
✅ Proper categories
✅ Featured flags
✅ Realistic dates and venues
